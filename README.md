# 🧠 Quiz App

A simple and interactive Quiz Application built using **HTML**, **CSS**, and **JavaScript**. This app allows users to test their knowledge through multiple-choice questions with instant feedback and score tracking.

---

## 🚀 Features

* 🎯 Multiple-choice quiz questions
* ⏱️ Instant answer validation
* 📊 Score tracking system
* 🔄 Restart quiz functionality
* 📱 Responsive design for mobile and desktop
* 🎨 Clean and user-friendly UI

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout design
* **JavaScript** – Logic, interactivity, and dynamic content

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html` or open it in your browser

---

## 🎮 How It Works

1. The quiz starts with a question displayed on the screen
2. Users select one of the given options
3. The app validates the answer instantly
4. The score updates based on correct answers
5. At the end, the final score is displayed
6. Users can restart the quiz to try again
